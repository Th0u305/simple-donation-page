What is the Document Object Model (DOM)?

The Document Object Model (DOM) is like a blueprint that represents the structure of a web page. It helps developers access and change different parts of the page, such as headings, paragraphs, images, and forms. Think of the DOM as a way to organize and control all the elements that make up a web page.

How do you select an element from the DOM?
  • getElementById()
  • getElementsByClassName()
  • getElementsByTagName()
  • querySelector()
  • querySelectorAll()
What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in JS that set event propagation (bubbling) to handle events at a higher level in the DOM than the element on which the event originated.Instead of adding individual event listeners to multiple child elements,one can add a single event listener to a parent element.This parent element can then manage all events that bubble up from its child elements.

How do you manipulate an element's attribute and styles using the DOM?

To manipulate HTML elements and their attributes using JavaScript we use the DOM (Document Object Model) function of JavaScript. The DOM represents the structure of your HTML document as a tree of objects, and you can use JavaScript to interact with these objects to change their properties, attributes, content, and more.
